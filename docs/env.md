# Таблица переменных

| Переменная         | Обязательна | Значение по умолчанию | Краткое описание                                    |
|--------------------|-------------|-----------------------|-----------------------------------------------------|
| `HTTP_PORT`        | нет         | `8080`                | Порт HTTP-сервера, на котором слушает сервис        |
| `DB_HOST`          | да          | `localhost`           | Хост PostgreSQL                                     |
| `DB_PORT`          | да          | `5432`                | Порт PostgreSQL                                     |
| `DB_USER`          | да          | `postgres`            | Пользователь базы данных                            |
| `DB_PASSWORD`      | да          | `postgres`            | Пароль пользователя базы данных                     |
| `DB_NAME`          | да          | `postgres`            | Имя базы данных                                     |
| `DB_SSLMODE`       | нет         | `disable`             | Режим SSL для подключения к PostgreSQL              |
| `POSTGRES_USER`    | да          | `postgres`            | Пользователь, создаваемый в контейнере PostgreSQL   |
| `POSTGRES_PASSWORD`| да          | `postgres`            | Пароль для `POSTGRES_USER`                          |
| `POSTGRES_DB`      | да          | `postgres`            | Имя создаваемой базы данных                         |
